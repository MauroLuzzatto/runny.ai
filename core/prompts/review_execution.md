Review how well I executed my recent runs. Evaluate **execution quality**, not just what was done.

**Per-run execution review** (for the most recent 5-10 runs):
- **Pacing consistency**: Were splits even, or did I fade (positive split) or start too slow (negative split)? Use lap data to assess.
- **Intensity compliance**: Was an easy run actually easy (HR in Z1-Z2)? Did hard sessions hit the right zone (Z4-Z5)?
- **HR drift**: Did heart rate climb at the same pace over the run? Flag cardiac drift >10% as a fatigue or heat signal.
- **Cadence & form**: Any unusual drops in cadence or changes in stride length that suggest fatigue?

**Overall execution patterns**:
- Are easy runs too fast?
- Are hard runs hard enough?
- Is there a consistent warm-up habit?
- Recovery between hard sessions: adequate or compressed?

**Scoring** (rate each reviewed run):
- Use a simple A/B/C grade: A = well executed, B = minor issues, C = needs attention
- Present as a table: Date | Type | Distance | Grade | Key Finding

End with 2-3 **actionable execution tips** I can apply immediately.
